# Unite Seoul 2026 - Unity AI와 MCP로 경험하는 프롬프트 기반 게임 개발

Unity AI 도구(Assistant, MCP, Generators)를 활용해 프롬프트만으로 게임을 제작하는 과정을 실습합니다.

![demo](demo.gif)

## Agenda

| 순서 | 세션 | 내용 |
| --- | --- | --- |
| 1 | **Unity AI Assistant와 Gateway** | AI Assistant 기본 사용법과 Gateway 연동 |
| 2 | **Unity AI MCP** | MCP를 통한 에디터 제어 및 자동화 |
| 3 | **Unity AI Generators** | 텍스처·사운드·메시 등 에셋 생성 |
| 4 | **Unity AI 게임제작 실습** | AI 도구를 조합한 게임 제작 핸즈온 |

## 개발 환경

- **Unity 6.4** 이후
- **Unity AI Assistant** 2.15.0-pre.2

## 사전 준비

### 1. Unity AI 계정 및 크레딧 확인

[Unity Cloud](https://cloud.unity.com) > Unity AI Assistant > 크레딧 관리에서 포인트를 확인합니다. 미가입자는 [Unity AI](https://unity.com/features/ai)에서 Free trial에 가입하세요.

### 2. 프로젝트 생성

Universal 3D (URP) 기본 템플릿으로 새 프로젝트를 생성합니다.

### 3. 패키지 설치

[AIWorkshopEssentials.unitypackage](https://drive.google.com/file/d/1sPFrn4W1PGiECAeICMeQv4EWss6WKqBf/view?usp=drive_link)를 다운로드해 프로젝트에 임포트합니다.

### 4. Unity AI Assistant 최신 패키지 설치

### 5. 옵션 설정

Unity > Preferences > AI 에서 다음과 같이 설정합니다.

- Enable Checkpoints — **Off**
- Enable Asset Knowledge — **On**
